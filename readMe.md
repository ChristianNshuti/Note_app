# Fullstack Project: Backend + Frontend

This repository contains a fullstack project with two separate folders:

- `backend` → Node.js server
- `frontend` → React frontend server

---

## Running Backend

### Install dependencies

```bash
cd backend
npm install

### Run the server
```
node server.js

---

## Running Frontend

### Install dependencies

```
cd frontend
npm install
```
### Run the server
```
npm run dev
```
