# Fullstack Project: Backend + Frontend

This repository contains a fullstack project with two separate folders:

- `backend` → Node.js server
- `frontend` → React frontend

---

## Running Backend

### Install dependencies

```bash
cd Backend
npm install 

### Run the server
```bash
node server.js

## Running Frontend

### Install dependencies

```bash
cd frontend
npm install

### Run the server
```bash
npm run dev
