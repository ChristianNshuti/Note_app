# Fullstack Project: Backend + Frontend

This repository contains a fullstack project with two separate folders:

- `backend` → Node.js server
- `frontend` → React frontend

---

## Running Backend

### Install dependencies

``
cd Backend
npm install
``

### Run the server
``
node server.js
``
## Running Frontend

### Install dependencies

``
cd frontend
npm install
``
### Run the server
``
npm run dev
``
